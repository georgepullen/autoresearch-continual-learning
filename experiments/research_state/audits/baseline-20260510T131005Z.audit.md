## What Changed

Compared with v10, v11 added current public fact-text LM replay and made final
consolidation target-biased:

- v10 visible target exact: 0.2396
- v10 visible anchor exact: 0.8924
- v11 visible target exact: 0.8021
- v11 visible anchor exact: 0.4375

This is the first smaller-model result showing that the 0.8B Qwen surrogate can
be a useful development lane when it trains edited facts as statements rather
than only exact prompt-answer pairs.

## Remaining Weakness

Anchor exact is above the surrogate floor but not comfortable. Relation-level
locality is still the obvious risk when graduating to the 4B champion lane. The
surrogate does not establish champion quality; it only justifies spending a
champion run on the same method family and data substrate.
