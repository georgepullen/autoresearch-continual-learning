## Diagnosis

The v9 result does not invalidate the wider Qwen surface by itself. It shows
that the v9 schedule made the wider surface unstable:

- rank-32 wide LoRA exposed substantially more capacity than v8
- learning rate was raised to `7.5e-4`
- final consolidation ended on target records

That combination collapsed generation into a single high-frequency edited
answer. The right follow-up is a stabilised wide-surface probe, not a further
capacity increase.

## Next Baseline

The v10 branch keeps the same prompt-disjoint CounterFact substrate and the same
wide surface, but changes optimizer pressure:

- rank-16 LoRA
- learning rate `1e-4`
- stronger anchor replay
- mixed final target/anchor consolidation
- no retrieval
- no answer-set postprocessor
- no visible-dev or locked-prompt rehearsal

Expected signature if this diagnosis is right:

- prediction entropy recovers versus all-`paris`
- public training-prompt recall beats v8's top-4 run
- visible-dev target/anchor exact gives a meaningful read on whether the smaller
  Qwen model is a useful surrogate
