## Debug Conclusion

The earlier failures were not caused by CounterFact being too toy-like or by the
0.8B surrogate being useless:

- top-4 attention-only Qwen LoRA underfit public training prompts
- wide rank-32 with high learning rate collapsed to a single frequent answer
- stable rank-16 wide LoRA memorized public prompts but initially lacked
  prompt-disjoint target transfer
- adding public fact-text LM replay provided the missing transfer signal while
  retaining enough anchor locality

This gives the in-loop Codex agent a substantially better starting champion:
the baseline is still simple sequential LoRA, but it is now on the intended
Qwen-family champion lane, uses a legitimate wider editable surface, and passes
the protected CounterFact standard confirmation without shims.
